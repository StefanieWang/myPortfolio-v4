About Me

I'm a self-taught front-end developer based in Brisbane, Australia. I love learning and coding. Programming has been always a fun activity for me and I'm really excited to learn new skills and technologies.

I started my web study journey with Ruby first, then quickly fell in love with coding. Coding in ruby is like writing an English story, natural and enjoyable.

Then I devote most of my time to frontend side of things. I learned HTML, CSS, Javascript, jQuery, Bootstrap, Git, etc. For me, JavaScript is filled with flexibility and surprising. The more I learn it the more I find its power. Currently I am learning ReactJS, and find the React way of thinking and building web apps is fun and inspiring.

There is still a long way ahead on my frontend study journey. I will keep learning and building new things and would love to dig into backend or any other tech to get the job done.

Roguelike Dungeon Crawler Game

✕

This is a roguelike dungeon crawler game built with React.

Player can beat enemies to gain XP and can pick up health items as well as weapons.

All dungeon maps are generated randomly. Players can move around within the map by pressing the arrow keys. Player enters the next dungeon through the gate entrance.

Game of Life

✕

The Game of Life is an example of a cellular automaton. Each square on board represents a cell (dead or alive). The status of a cell in the next generation is determined by a set of rules.

In this game of life, user can choose from different board sizes, sim speed, and simple patterns.

User can observe and investigate the evolution of cells by pause, step running or adding new cells anytime.

React JS - JavaScript - jQuery - Canvas - HTML - CSS

Simon Game

✕

Simon is a classic electronic game of memory skill. It creates a series of tones and lights and requires a user to repeat the series. If the user succeeds the series becomes progressively longer and more complex.

Users can restart the game anytime by hit the "on-off" button.

Users can choose to play in strict mode where if they get a button press wrong, the game will restart to a single step.

User is notified the number of steps in the current series.

JavaScript - jQuery - HTML - CSS
